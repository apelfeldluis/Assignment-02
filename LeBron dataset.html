Section 1: Description of the Data

The LeBron dataset from FiveThirtyEight includes statistics for LeBron James, collected from his basketball games. It measures different performance metrics, like game scores and player stats, over various periods. The data is saved in a CSV file.

Section 2: Reading the Data into R


library(readr)
url <- "https://raw.githubusercontent.com/fivethirtyeight/data/master/lebron/lebron.csv"
lebron_data <- read_csv(url)

Section 3: Cleaning the Data


library(dplyr)
cleaned_lebron_data <- lebron_data %>%
  rename(GameID = game_id, Points = pts, Assists = ast) %>%
  filter(!is.na(GameID))
head(cleaned_lebron_data)

Section 5: Summary Statistics


summary_data <- cleaned_lebron_data %>% select(GameID, Points, Assists)
summary_stats <- summary_data %>% summarize(
    MinPoints = min(Points, na.rm = TRUE),
    MaxPoints = max(Points, na.rm = TRUE),
    MeanPoints = mean(Points, na.rm = TRUE),
    MissingPoints = sum(is.na(Points)),
    MinAssists = min(Assists, na.rm = TRUE),
    MaxAssists = max(Assists, na.rm = TRUE),
    MeanAssists = mean(Assists, na.rm = TRUE),
    MissingAssists = sum(is.na(Assists))
)
